# 🐸 蛙田文化官网设计方案

## 技术栈
- 纯前端单页应用（单 HTML 文件）
- p5.js 1.11.3 做动态背景 + 交互动画
- CSS 动画 + Intersection Observer 做滚动展示
- 响应式设计（手机/平板/桌面）
- 无后端，可部署到 Cloudflare Pages / Vercel 免费托管

## 设计语言
- **主色调:** 青碧色（蛙田意象）+ 暖金色（文化气质）
- **风格:** 现代东方美学，简约但有层次
- **字体:** Noto Sans SC（思源黑体）+ Noto Serif SC（思源宋体）

## 页面结构
```
导航栏（固定顶部，玻璃效果）
├── 首页 — 全屏 Hero + p5.js 流动粒子动画
│   └── 品牌故事 + 五大板块快速入口
├── 蛙田美育 — 艺术教育，暖色粒子 + 画笔元素
├── 蛙田音乐 — 音乐可视化律动
├── 蛙田商城 — 产品展示卡片
├── 蛙田乐器 — 乐器精致展示
├── 蛙田科技 — 科技感数据流动
├── 品牌故事 — 公司介绍
└── 底部 — 联系信息 + 版权
```

## p5.js 动态方案
- 全局一张画布作为背景
- 随滚动位置切换粒子颜色和运动风格
- 首页：流动的青色粒子（蛙田意象）
- 各板块切换时色彩渐变过渡
